# Components

Get to know Bowline's React components.

- Control nearly every aspect with a well-defined theme configuration
- Map multiple design tokens to your theme's breakpoints with responsive props
- Straightforward layouts with Stack, Inline, and Columns
- Auto-detected contrast ratio adapts text and other component styles to the current background color
